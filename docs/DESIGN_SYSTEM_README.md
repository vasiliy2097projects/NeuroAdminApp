# NeuroAdmin Design System

## Обзор

Дизайн-система NeuroAdmin - это комплексная библиотека компонентов, стилей и руководящих принципов для создания консистентного пользовательского интерфейса платформы NeuroAdmin.

**Версия:** 1.0  
**Основа:** Material Design + Material-UI  
**Статус:** В разработке

## 🎯 Основные принципы

### 1. Консистентность

Все компоненты следуют единым принципам дизайна и поведения.

### 2. Доступность

Все компоненты соответствуют WCAG 2.1 стандартам (минимум AA уровень).

### 3. Адаптивность

Дизайн работает на всех устройствах (mobile, tablet, desktop).

### 4. Производительность

Компоненты оптимизированы для быстрой загрузки и отзывчивости.

### 5. Масштабируемость

Дизайн-система легко расширяется новыми компонентами.

## 🎨 Цветовая палитра

### Основные цвета

- **Primary:** `#2196F3` (Синий) - Основные действия, CTA
- **Secondary:** `#9C27B0` (Фиолетовый) - Вторичные действия, акценты

### Статусы

- **Success:** `#4CAF50` (Зеленый) - Успешные операции
- **Warning:** `#FF9800` (Оранжевый) - Предупреждения
- **Error:** `#F44336` (Красный) - Ошибки
- **Info:** `#2196F3` (Синий) - Информация

Подробнее в [Style Guide](./STYLE_GUIDE.md#-цветовая-палитра).

---

## 📝 Типографика

### Шрифты

- **Основной:** Roboto, Inter, system fonts
- **Моноширинный:** Roboto Mono (для кода)

### Размеры

- **H1:** 40px (2.5rem) - Главные заголовки
- **H2:** 32px (2rem) - Заголовки разделов
- **H3:** 28px (1.75rem) - Подзаголовки
- **Body:** 16px (1rem) - Основной текст
- **Caption:** 12px (0.75rem) - Подписи

Подробнее в [Style Guide](./STYLE_GUIDE.md#-типографика).

## 🧩 Компоненты

### Базовые компоненты

- **Button** - Кнопки (Primary, Secondary, Outlined, Text)
- **Input** - Поля ввода (Text, Password, Email, Number)
- **Card** - Карточки для контента
- **Table** - Таблицы данных
- **Modal** - Модальные окна
- **Badge** - Бейджи и статусы
- **Progress** - Прогресс-бары
- **Loading** - Индикаторы загрузки
- **Alert** - Уведомления
- **Select** - Выпадающие списки
- **Checkbox** - Чекбоксы
- **Radio** - Радиокнопки
- **Tabs** - Вкладки

### Layout компоненты

- **Header** - Шапка сайта
- **Sidebar** - Боковая панель навигации
- **Footer** - Подвал
- **Container** - Контейнер контента
- **Grid** - Сетка (12 колонок)

Полные спецификации в [Компоненты](./COMPONENTS_SPEC.md).

## 📱 Адаптивность

### Breakpoints

- **xs:** 0px - Мобильные
- **sm:** 600px - Планшеты (портрет)
- **md:** 960px - Планшеты (альбом)
- **lg:** 1280px - Десктоп
- **xl:** 1920px - Большие экраны

### Grid система

12 колонок с gutter 16px

---

## ♿ Доступность

### Контрастность

- Минимум 4.5:1 для обычного текста (AA)
- Минимум 3:1 для крупного текста (AA)
- Рекомендуется 7:1 для лучшей доступности (AAA)

### Keyboard Navigation

- Все интерактивные элементы доступны с клавиатуры
- Логический порядок Tab
- Focus indicators видимы

### ARIA

- Все компоненты имеют правильные ARIA атрибуты
- Screen reader friendly
