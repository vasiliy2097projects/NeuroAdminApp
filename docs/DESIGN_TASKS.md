### Задачи

#### 4.1 Bot Detection Main Page
- [ ] Wireframe структуры
- [ ] Визуальный дизайн:

    - Выбор социальной сети (VK/OK)
    - Выбор аккаунта

  - Список анализов:
    - Карточки анализов с информацией
    - Статусы (Pending, InProgress, Completed, Failed)
    - Прогресс-бар для активных анализов
    - Фильтры и сортировка
- [ ] Состояния:

  - Загрузка

- [ ] Адаптивный дизайн

#### 4.2 Analysis Details Page
- [ ] Дизайн страницы деталей анализа
- [ ] Информация об анализе:
  - Статус
  - Прогресс
  - Дата создания/завершения
  - Количество проверенных пользователей
- [ ] Кнопка просмотра результатов
- [ ] Кнопка отмены (для активных анализов)

#### 4.3 Analysis Results Page
- [ ] Дизайн страницы результатов
- [ ] Таблица ботов:
  - Колонки: ID пользователя, Имя, Bot Score, Статус, Действия
  - Фильтры (по статусу, по score)
  - Поиск
  - Пагинация
  - Сортировка
- [ ] Статистика:
  - Общее количество проверенных
  - Количество ботов
  - Процент ботов
  - График распределения scores
- [ ] Действия:
  - Экспорт результатов (CSV, PDF)
  - Массовые действия
- [ ] Модальное окно с деталями бота:
  - Фото профиля
  - Информация о пользователе
  - Детали анализа (почему определен как бот)
  - Bot Score breakdown

#### 4.4 Компоненты для модуля
- [ ] Карточка анализа (AnalysisCard)
- [ ] Прогресс-бар анализа
- [ ] Статус-бейджи

- [ ] Фильтры и поиск

- [ ] График распределения Bot Scores
- [ ] Круговая диаграмма (боты/не боты)
