# NeuroAdmin Roadmap

## Обзор

Дорожная карта разработки NeuroAdmin - AI-powered SMM платформы с модулем очистки от ботов.

---

## 🎯 Цели проекта

1. Создать работающую систему аутентификации
2. Реализовать интеграции с VK и OK API
3. Разработать модуль очистки от ботов
4. Подготовить проект к production deployment

---

## 🚀 Release 1.0: Базовая инфраструктура

**Цель:** Создать основу для дальнейшей разработки - работающая система аутентификации и базовый фронтенд.

**Срок:** 4-6 недель

### Backend

- ✅ API Gateway Service (#5)

  - Базовая структура и проксирование
  - JWT валидация
  - Интеграция с другими сервисами

- ✅ Auth Service (#6)
  - Регистрация и вход
  - JWT токены
  - Управление пользователями

### Frontend

- ✅ Frontend Setup & Infrastructure (#7)

  - React + TypeScript + Vite
  - Redux Toolkit
  - Material-UI
  - API клиент

- ✅ Frontend Authentication (#8)
  - Страницы Login/Register
  - Управление профилем
  - Защищенные маршруты

### Design

- 🔄 Дизайн-система и UI компоненты (#17)
- 🔄 Дизайн страниц аутентификации (#18)
- 🔄 Дизайн Dashboard (#19)

**Критерии готовности:**

- [ ] Пользователь может зарегистрироваться и войти
- [ ] JWT токены работают корректно
- [ ] Frontend может общаться с API Gateway
- [ ] Базовая навигация работает

## 🤖 Release 3.0: Основной функционал - очистка от ботов

**Цель:** Реализовать полный функционал модуля очистки от ботов с UI.

**Срок:** 6-8 недель

### Backend

- ⏳ Bot Detection Service - синхронные операции (#11)

  - Создание анализов
  - Получение статуса и результатов
  - API endpoints

- ⏳ Bot Detection Service - асинхронные операции (#12)
  - RabbitMQ интеграция
  - Background workers
  - Стратегии определения ботов
  - Интеграция с VK/OK Services

### Frontend

- ⏳ Frontend Bot Detection Module (#13)
  - Страница запуска анализа
  - Список анализов
  - Страница результатов
  - Визуализация данных

### Design

- 🔄 Дизайн модуля очистки от ботов (#20)

**Критерии готовности:**

- [ ] Можно запустить анализ ботов
- [ ] Анализ обрабатывается асинхронно
- [ ] Результаты отображаются в UI
- [ ] Стратегии определения ботов работают
- [ ] Можно экспортировать результаты

## 📝 Заметки

- Все дизайн задачи должны быть завершены до начала разработки соответствующих frontend компонентов
- Backend сервисы могут разрабатываться параллельно
- Интеграционные тесты должны покрывать взаимодействие между сервисами
- Документация обновляется по мере разработки
