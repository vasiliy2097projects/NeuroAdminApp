# NeuroAdmin

**AI-Powered SMM Platform** — интеллектуальная платформа для автоматизации управления социальными медиа на базе искусственного интеллекта.

## 🎯 О проекте

NeuroAdmin — это комплексная SMM-платформа, использующая технологии искусственного интеллекта для автоматизации задач управления социальными сетями. Платформа разрабатывается модульно, позволяя постепенно расширять функциональность.

## 🚀 Текущая версия

**Модуль очистки аудитории от ботов**

Первая реализуемая функция платформы — автоматическая идентификация и очистка аудитории от ботов, фейковых и неактивных аккаунтов. Модуль использует интеллектуальные алгоритмы для анализа поведения пользователей, паттернов активности и других метрик для точного определения ботов.

### Поддерживаемые платформы:

- **ВКонтакте (VK)** — интеграция через VK API
- **Одноклассники (OK)** — интеграция через OK API

### Ключевые возможности:

- 🤖 Автоматическое обнаружение ботов и фейковых аккаунтов
- 📊 Анализ активности и поведения пользователей
- 🎯 Интеллектуальная фильтрация аудитории
- 📈 Улучшение качества и вовлеченности аудитории
- 🔌 Интеграция с социальными сетями через официальные API

## 🏗️ Архитектура

Проект построен на **микросервисной архитектуре** с использованием MVC паттерна.

### Структура микросервисов:

```
NeuroAdminApp/
├── services/
│   ├── api-gateway/              # API Gateway (BFF)
│   ├── auth-service/             # Сервис аутентификации
│   ├── bot-detection-service/    # Сервис очистки от ботов
│   ├── vk-service/               # Интеграция с VK API
│   └── ok-service/               # Интеграция с OK API
├── frontend/                     # React + TypeScript
└── docker-compose.yml
```

### Коммуникация между сервисами:

- **HTTP REST** — для синхронных операций (получение статуса, результатов)
- **RabbitMQ** — для асинхронных операций (обработка анализа ботов)

## 🛠️ Технологический стек

### Backend

- **.NET 8** — платформа разработки
- **ASP.NET Core MVC** — веб-фреймворк
- **PostgreSQL** — база данных
- **Dapper** — микро-ORM для работы с БД
- **NLog** — логирование
- **Refit** — декларативные HTTP клиенты
- **Polly** — resilience patterns (retry, circuit breaker)
- **RabbitMQ** — очереди сообщений
- **Redis** — кэширование

### Frontend

- **React 18** — UI библиотека
- **TypeScript 5** — типизированный JavaScript
- **Vite** — сборщик
- **Redux Toolkit** — управление состоянием
- **Material-UI** — UI компоненты
- **React Hook Form + Zod** — работа с формами и валидация
- **Axios** — HTTP клиент

### Infrastructure

- **PostgreSQL 15** — база данных
- **Redis 7** — кэширование
- **RabbitMQ 3.12** — очереди сообщений
- **Docker** — контейнеризация

## 📋 Быстрый старт

### Требования

- .NET 8 SDK
- Node.js 18+
- Docker и Docker Compose
- PostgreSQL 15+ (или через Docker)

### Запуск через Docker Compose

```bash
# Клонировать репозиторий
git clone <repository-url>
cd NeuroAdminApp

# Запустить все сервисы
docker-compose up -d
```

### Локальная разработка

#### Backend сервисы

```bash
# API Gateway
cd services/api-gateway
dotnet restore
dotnet run

# Auth Service
cd services/auth-service
dotnet restore
dotnet run

# Bot Detection Service
cd services/bot-detection-service
dotnet restore
dotnet run

# VK Service
cd services/vk-service
dotnet restore
dotnet run

# OK Service
cd services/ok-service
dotnet restore
dotnet run
```

#### Frontend

```bash
cd frontend
npm install
npm run dev
```
