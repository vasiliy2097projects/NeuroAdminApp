# NeuroAdminApp

Административное приложение для управления нейросетевыми сервисами.

## Структура проекта

```
NeuroAdminApp/
├── frontend/          # React frontend приложение
├── backend/           # Backend сервисы
└── README.md
```

## Технологии

- **Frontend**: React, React Router
- **Backend**: Spring Boot (микросервисы)

## Быстрый старт

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
mvn clean install
mvn spring-boot:run
```
